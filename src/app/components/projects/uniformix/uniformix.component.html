Introdução

O principal objetivo do Uniformix é fornecer controle de inventário de uniformes da empresa, permitindo que as operações sejam registradas por meio de um histórico de transações e fisicamente através de protocolos assinados pelos funcionários.

Requisitos levantados

ID Requisitos Funcionais
RF-01 O sistema deve permitir as operações de criação, leitura, edição e exclusão de lotes
RF-02 Os lotes devem conter as seguintes informações: descrição, quantidade, categoria, fornecedor e data de aquisição
RF-03 O sistema deve permitir as operações de criação, leitura e edição de uniformes/EPIs
RF-04 Os uniformes/EPIs devem conter as seguintes informações: descrição, quantidade, sexo(gênero) e tamanho
RF-05 O sistema deve conter uma janela de “Configurações” para criar categorias, fornecedores e unidades (lojas/departamentos).
RF-06 O usuário deve poder exportar os dados dos lotes registrados
RF-07 O sistema deve registrar um log de transações com pesquisa por nome do colaborador ou loja
RF-08 O usuário deve poder exportar os dados de transações realizadas, filtradas por loja e período
RF-09 O sistema deve possuir uma janela de entrada/saída para realizar operações de entrega ou devolução de fardamento, com as seguintes informações: nome do colaborador, loja/departamento(Unidade), quantidade e data de entrega.
RF-10 O sistema deve gerar um protocolo disponível para impressão com as seguintes informações: termo de compromisso, nome do colaborador, quantidade entregue, código do protocolo, data de emissão e campo para assinatura.

Funcionalidades

  • Operações CRUD: Criar, ler, atualizar e excluir uniformes, lotes de uniformes, unidades, categorias e fornecedores.
  • Geração de Relatórios: Relatório de transações, uniformes e lotes.
  • Emissão de Protocolos: Para cada transação, o sistema emite um protocolo em PDF para impressão e assinatura física.
  • Interface Web
  • Autenticação de Usuário
  • Hashing e Salting seguro de senhas usando BCrypt.
  • Integração com o Banco de Dados PostgreSQL para armazenamento de informações do sistema.
