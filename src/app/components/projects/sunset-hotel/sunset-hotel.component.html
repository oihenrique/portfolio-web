Introdução

A ideia principal por trás do projeto Sunset Hotel foi criar um sistema de reservas que permita gerenciar reservas de hotel e clientes ativos, bem como quartos e usuários do sistema.

O projeto faz parte do curso Oracle One Next, oferecido pela Alura, e tem como objetivo implementar o conceito CRUD usando ORM com Hibernate, conectando-se a um banco de dados MySQL, e desenvolvendo interfaces gráficas usando Swing.

Tecnologias utilizadas

  • Java: utilizado para construção do backend e conexão JDBC com banco de dados.
  • Java Swing: utilizado para criação da interface de usuário.
  • Hibernate: utilizado para conexão entre o backend e o banco de dados.
  • MySQL: banco de dados relacional para armazenamento dos dados.

Funcionalidades

  • Operações CRUD: Registro de usuários, reservas, hóspedes e quartos. Além disso, edição e exclusão de usuários, reservas, hóspedes e quartos.
  • Interface desktop de Usuário para Uso do Sistema.
  • Login de Usuário
  • Hashing e Salting Seguro de Senhas usando BCrypt.
  • Integração com Banco de Dados MySQL para armazenamento das informações do sistema.
